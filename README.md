chat
=====

An OTP + Cowboy websocket application

Build
-----
    git clone https://github.com/abeniaminov/erlelmchat.git
    cd erlelmchat
    ./rebar3 get_deps
    ./rebar3 compile
    ./chat.sh start or ./chat.sh startd 
    
    http://localhost:8001
    
    Fill user name then push Join button.
    Fill free to send messages
